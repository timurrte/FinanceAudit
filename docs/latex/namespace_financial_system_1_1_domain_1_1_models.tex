\doxysubsection{Detailed Description}
Абстрактний базовий клас для всіх банківських рахунків. Не може бути створений безпосередньо. Реалізує інкапсуляцію фінансових даних. 

Транзакція списання коштів (Спадкоємець 2).

Базовий абстрактний клас для всіх фінансових транзакцій. Реалізує інтерфейс ITransaction\+Processor.

Ощадний рахунок. Нараховує відсотки на позитивний залишок. Успадковує базову логіку від \doxylink{class_financial_system_1_1_domain_1_1_models_1_1_bank_account}{Bank\+Account}.

Транзакція зарахування коштів (Спадкоємець 1).

Клас, що представляє клієнта банку. Демонструє відношення Агрегації з \doxylink{class_financial_system_1_1_domain_1_1_models_1_1_bank_account}{Bank\+Account} (1 до багатьох).

Кредитний рахунок. Баланс представляє суму боргу.

Розрахунковий рахунок. Дозволяє балансу опускатися нижче нуля в межах ліміту овердрафту. Стягує щомісячну комісію за обслуговування.

Базовий абстрактний клас сутності. Реалізує інтерфейс IEntity.
